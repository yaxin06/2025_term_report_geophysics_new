🚀 建立 Pull Request (PR) 繳交作業

  1. 回到你的 GitHub 頁面，點擊 Contribute → Open pull request。
  2. 確認左側 Base 為老師的 Repo，右側 Head 為你的 Fork Repo。
  3. 重點： 仔細填寫標題與內容，附上你的 ObsPy/PyGMT 結果圖，並寫下你的觀察。
  4. 點擊 Create pull request，看到「Open」標籤即代表繳交成功！

🔔 批改與回饋

01 實驗目的與設備

本次實驗旨在探測從黑森林至圖書館後方的地層結構。我們利用人工擊發震波，測量地層厚度及波速（V1, V2）。

  • 24 個 Geophones (間距 2m, 全長 48m)
  • 鐵槌敲擊鐵板作為震源
  • 數位走時分析電腦系統
圖一：黑森林實驗現場佈置
圖二：檢波器回傳之原始波相

02 數據採集與分析

透過敲擊不同炸點（0m, 6m, 12m...），記錄震波到達各檢波器的時間。如圖所示，我們可以觀察到波型的相位變化，進而判讀初至波 (First Arrival)。

「24 個檢波器的設計是因為其因數眾多，方便我們在分段取距與計算平均值時具有極佳彈性。」

03 走時曲線與速度推算

將數據繪製成 Time-Distance 圖。斜率倒數即代表波速。我們成功分離出：

直達波 (V1) 平均 333.01 m/s
折射波 (V2) 平均 813.13 m/s
圖三：走時曲線斜率分析
圖四：最終數據彙整統計

04 數據校正與結論

在分析過程中，我們秉持科學嚴謹度，剔除了炸點 12m 與 42m 的數據。這兩組測量值展現了明顯的異常（厚度最大與最小），可能源於地表局部雜訊或耦合不良。

最終結論

推算地層厚度為 7.67 公尺。這項結果揭示了校園地底速度不連續面的深度，為後續的地質研究提供了堅實的基底數據。
