監測的雙重目標

短期監測： 於破壞性 S 波到達前發出警報，爭取逃生時間。

長期監測： 透過大量微小地震資料累積，拼湊出「盲斷層」的幾何形狀。

AI 的介入點

傳統人工判讀波相在面對大量餘震（如花蓮地震）時，不僅費時且具備主觀偏差。透過 Deep Learning 的抗噪能力，AI 能在人類聽不見的雜訊中，精確挑選出 P 波相位。

火山物理學：沉與浮的奧祕

火山渣與浮石的差異在於氣孔構造。火山渣氣孔連通，入水即沉；浮石氣孔獨立形成空氣室，因而能漂浮。每一顆石頭都記錄了岩漿噴發那一瞬的冷卻壓力故事。

防災與地磁觀測

日本利用地磁變化來觀測岩漿位置。當岩石被加熱超過居禮溫度時會消磁，若觀測到地磁減弱，則暗示地下岩漿正在上升。此外，日本將「震災關聯死」（如避難壓力導致的健康問題）納入統計，展現出更深厚的人文照護思維。
